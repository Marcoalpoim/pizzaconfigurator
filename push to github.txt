HOW TO RUN IT? 

🍕 Pizza Configurator — Setup Guide

1. 📦 Requirements

            Before anything, make sure you have:

            Node.js (recommended: v18+)
            npm (comes with Node)

            Check if installed:

            node -v
            npm -v

2. 📥 Download the Project

            Option A — ZIP
            Download the project
            Extract it
            Option B — Git
            git clone <your-repo-url>
            cd your-project-folder

3. 📦 Install Dependencies

            Inside the project folder:

            npm install

            This installs things like:

            React
            Three.js
            other required packages

4. ▶️ Run the Project

            npm run dev

            or (depending on setup):

            npm start
